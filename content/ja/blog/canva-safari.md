iPad で Canva と Safari を Split View で使うには、両アプリを開いた状態でマルチタスクメニューから Split View を選ぶか、Splicon で作成したホーム画面ショートカットをタップする方法が最も手軽です。

Canva と Safari を横並びで表示すると、デザイン作業中に参考サイトやブランドカラーをすぐ確認できるため、アプリの切り替え回数が大幅に減ります。特に次のような場面で役立ちます。

- **参考画像の収集**: Safari で Pinterest やブランドサイトを開きながら、Canva のキャンバスに合う配色やレイアウトをリアルタイムで見比べる
- **コピー&ライティング**: Safari でリサーチした文章やキャッチコピーの例を見ながら、Canva のテキストボックスに直接入力する
- **フォント・カラーコードの参照**: Safari でブランドガイドラインのページを表示し、カラーコードを Canva のカラーピッカーにそのまま入力する

## Splicon でセットアップする方法

まだアプリを持っていない方は、まず [Splicon を App Store から無料でダウンロード](https://apps.apple.com/us/app/splicon-split-view-icon-maker/id6762535752) してください。

1. **Splicon を開き、Canva と Safari を選ぶ**: アプリ内の検索欄でそれぞれのアプリを検索し、Split View のペアとして設定します。
2. **アイコンを生成して保存**: 好みのスタイルを選んでアイコンを生成し、写真アプリに保存します。
3. **ショートカットアプリで設定**: ショートカットアプリで新規ショートカットを作成し、Split View を起動するアクションを追加。アイコンには先ほど保存した画像を指定します。
4. **ホーム画面に追加**: ショートカットをホーム画面に追加すれば完了です。次回からはアイコンをタップするだけで Canva と Safari が並んで起動します。

この組み合わせを一度ホーム画面に置いておくと、デザイン作業を始めるたびに手順を繰り返す必要がなくなり、作業の流れを止めずに集中できます。
