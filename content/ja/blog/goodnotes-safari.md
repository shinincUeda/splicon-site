iPad で GoodNotes と Safari を Split View で使うには、画面を左右に分割して両アプリを同時に表示することで、調べながら書くという作業をひとつの画面で完結できます。

GoodNotes と Safari を横に並べると、次のような場面で力を発揮します。

- **リサーチノート作成**: Safari で開いた記事や論文を読みながら、GoodNotes に要点を手書きでまとめられます。コピー＆ペーストで引用元をそのまま貼り付けることも可能です。
- **語学学習**: ブラウザで英文記事を読みながら、GoodNotes に単語や例文をメモしてオリジナル単語帳を作れます。
- **授業・講義の予習復習**: 配布資料のURLやオンライン教材を Safari で開き、GoodNotes のノートに書き込みながら学習内容を整理できます。

ただし、毎回 Split View を手動で設定するのは手間がかかります。そこで Splicon を使うと、この組み合わせをホーム画面のアイコンにしておき、タップ一発で起動できるようになります。

## Splicon でセットアップする方法

まだアプリをお持ちでない方は、[Splicon を App Store から無料でダウンロード](https://apps.apple.com/us/app/splicon-split-view-icon-maker/id6762535752)してから以下の手順を進めてください。

1. **Splicon を開き、GoodNotes と Safari を選ぶ**: アプリ内の検索欄でそれぞれのアプリを探し、Split View のペアとして設定します。
2. **アイコンを生成して保存する**: スタイルやデザインを選択してアイコンを生成し、写真アプリに保存します。
3. **ショートカットアプリで設定する**: ショートカットアプリで新規ショートカットを作成し、Split View を開くアクションを追加。ショートカットのアイコンに Splicon で保存した画像を設定します。
4. **ホーム画面に追加する**: ショートカットをホーム画面に追加すれば完成です。

設定は一度だけ。次回からはアイコンをタップするだけで GoodNotes と Safari が並んで起動します。
